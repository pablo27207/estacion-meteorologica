# Firmware Directory

This directory stores firmware binaries for OTA updates.

## Files

- `version.json` - Current firmware version metadata
- `firmware_tx.bin` - TX node firmware binary (generated by PlatformIO)

## Updating Firmware

1. Build new firmware: `pio run -e tx_node`
2. Copy `.pio/build/tx_node/firmware.bin` to this directory as `firmware_tx.bin`
3. Update `version.json` with new version number and file size
4. Optionally compute MD5: `certutil -hashfile firmware_tx.bin MD5`

## version.json Format

```json
{
  "version": "1.0.0",
  "filename": "firmware_tx.bin",
  "description": "Brief description of changes",
  "date": "YYYY-MM-DD",
  "minVersion": "0.0.0",
  "size": 123456,
  "md5": "abc123..."
}
```

- `minVersion`: Minimum version required to upgrade (for breaking changes)
